# NodeJS Backend
>Backend API for a NodeJS application,which is a Devcamper directory website

## Usage
Rename "config/config.env.env" to "config/config.env" and update the values/setting to your own.

## Install Dependencies
```
npm install
```
## Run App
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
